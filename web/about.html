Shadow Lab — Microstructure Engine

A live order-book microstructure terminal. This is a public demo.

This is a research lab, not a fund. Everything here runs on a synthetic L3 order-book stream (deterministic). The methodology is real; the displayed numbers are illustrative and are not live trading performance.

Architecture

A deterministic synthetic Level-3 feed is replayed through the same production detection pipeline used on real data, then streamed to the browser over a WebSocket.

  synthetic L3 feed ─▶ 6-layer detection pipeline ─▶ /ws (30 Hz) ─▶ browser
   (ingestion/         · cup-flip state machine      DepthIndicatorFrame   chart + DOM
    synthetic.py)      · spoof / authenticity                              + overlays
                       · staircase fragility                               + exec sim
                       · spectral / institutional force                    (paper)
                       · level tracker / DOM
                       · regime context → weights

Scripted events (momentum runs, institutional walls, spoofs, stop-runs) fire on a 60-second loop so every detector layer is visible.
