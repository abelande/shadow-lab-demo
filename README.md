# Shadow Lab — Microstructure & Research Demos

Public, stripped demos derived from a private research stack. **No live data, no
credentials, no proprietary models** — everything here runs on synthetic or sample data.

- **Microstructure engine** — a synthetic L3 order-book feed with live overlays
  (level lifecycle, spoof / cup-flip / regime badges), a replay scrubber, and a paper exec sim.
- **`research/` (p6lab)** — research notebooks: feature engineering, event / triple-barrier
  labeling, and leakage-free purged cross-validation (CPCV), rendered to static HTML.

> Status: scaffolding — build in progress. Methodology is real; displayed numbers are illustrative.
